Shortcuts

1. Open Side Menu

To open the Side Menu with the following key shortcut, you must not have any other window open in the app.

Ctrl + M

2. Close Side Menu

To close the Side Menu with the following key shortcut, the side menu should be open first.

Shift + M

3. Switch Dark/Light Mode

Use the first Shortcut to switch to darkmode and the second for lightmode. Use the third key to toggle between light and darkmode.

Shift + D
Shift + L
Shift + T

4. Add new index card to the stack

This shortcut only works if the front and back of the new card are labeled and the focus is setted to the back of the card.

Enter

5. Switch to front and back

Only works if one of the fields from the index card is focused.

key down / key up

6. Open Playground

To open the Playground with the following key shortcut, you must not have any other window open in the app.

Shift + P

7. Close Playground

To open the Playground with the following key shortcut, you must not have any other window open in the app.

Shift + G
